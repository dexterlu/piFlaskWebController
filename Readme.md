# Readme
Web Simple Button control Rpi3 GPIO controller

### Requirement
- pre-install requirements
```
sudo pip install -r requirements.txt
```

### Runing
```
python3 main.py
```
and then visit:
```
http://[your Rpi3's IP Address]:5678
```
